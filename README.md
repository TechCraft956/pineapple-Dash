# Pineapple OS

Pineapple OS is a local operator system that turns runtime truth into next actions.

## Strongest surfaces

- **Pineapple-Dash**: canonical operator surface for runtime health, approvals, opportunities, executions, and Copilot
- **Canonical runtime state**: shared truth under `../pineapple-ops-runtime`
- **Marketplace engine**: ingestion, scoring, ranking, and operator briefings
- **Copilot panel**: visible intelligence layer that explains what to do next and why

## Demo path

1. Open `Overview`
2. Open `System Health`
3. Open `Approvals`
4. Open `Opportunities`
5. Open `Copilot`

## Core idea

This is not a dashboard-first system.
It is a runtime truth -> interpretation -> action system.

## Canonical state

Runtime outputs live in:

## Best public signal

What matters externally:

- truthful service health
- canonical approval path
- file ingestion with extracted context
- generated tasks from live state
- visible copilot recommendations
